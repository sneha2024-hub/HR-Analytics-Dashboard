## 📊 HR Analytics Dashboard: Employee Attrition Analysis

## 🔍 Project Overview

This project analyzes employee attrition patterns using Power BI. The dashboard helps identify key factors influencing employee turnover such as salary, job role, and department.

## 🛠 Tools Used

* Power BI
* DAX
* Data Modeling

## 📈 Key Features

* KPI cards: Attrition Rate, Total Employees, Employees Left, Avg Salary, Avg Age
* Interactive slicers (Department, Gender)
* Attrition analysis by salary, job role, department, and gender

## 💡 Key Insights

* Higher attrition observed in lower salary ranges
* Certain job roles show higher turnover
* Department-wise variation in employee retention

## 🚀 Conclusion

This dashboard provides actionable insights to help HR teams improve employee retention strategies.
